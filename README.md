VidVibe – Teacher Web Application

VidVibe is a web-based platform designed to enhance teacher-student interaction by providing a structured and engaging digital environment for academic communication and content delivery.

Developed as part of a hackathon project, the application focuses on simplifying classroom management by allowing teachers to efficiently share resources, manage content, and interact with students in a centralized system.

🔹 Key Features:

User-friendly interface for teachers and students
Organized content sharing and communication system
Responsive web design for accessibility across devices

🔹 Tech Stack:

HTML
CSS
JavaScript
